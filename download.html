Download

All data published on this site are open source. You can download a copy from a repository hosted on GitHub.

$ git clone https://github.com/onionltd/oniontree

Bookmarks

Download and import OnionTree Bookmarks in your Tor Browser to have immediate access to all popular onion services. For a technical how to, please refer to Import Bookmarks from an HTML file.

If you are a Tails user, don't forget to configure your persistent volume to preserve the bookmarks between reboots.
