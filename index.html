Welcome!

Onion Limited's Oniontree is a repository of Tor hidden services. Data are licensed under CC0 1.0, distributed in a machine readable format.

This repository is work of dedicated volunteers. We are not affiliated with any of the services listed here, nor we provide any type of endorsement. This work is published in hope it will be useful for security researchers, journalists and anyone who needs machine-readable data.

Use this site to find onion addresses, PGP keys and other information about Tor hidden services.
